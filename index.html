Como usar

1. HTML

Você precisa de um elemento pai que será o elemento onde o evento do movimento do mouse será inserido. Isso quer dizer que, quando o mouse passar sobre este elemento, as imagens indicadas pelo parâmetro elements serão animadas.

2. Javascript

Você deve chamar a função moover no elemento pai, desta forma, o evento mouseMove será inserido nele. Nos parâmetros da função, você deve passar um array chamado elements com os elementos que serão animados.
